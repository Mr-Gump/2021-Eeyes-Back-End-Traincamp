# `Python`之禅

##### 打印`Python`之禅,需要以下几步：

1. 打开`Windows Terminal`或按<kbd>win</kbd>+<kbd>R</kbd>打开运行，再输入`cmd`打开`Command`窗口 

2. 输入`Python`,进入`Python`的交互模式

3. 输入以下命令

   ```python
   import this
   ```

4. 这时你将成功看到**大名鼎鼎**的`Python`之禅
