# markdown自学觉得有用并掌握的语法

1. 代码框

   ```markdown
   ​```（或‘~~~’）（空格）
   代码
   不选用语言可用于强调内容
   ```

2. 标题

   ```markdown
   # 一级标题
   ## 二级标题
   ……
   ###### 六级标题
   ```

3. 分割线

   ```
   · 标题下自带
   · ***
   · ---
   · +++
   ```

4. 加重标识

   ```markdown
   `阴影`
   *斜体*
   **加粗**
   ****加粗斜体****
   ~~横线批掉~~
   <kbd>键盘按钮</kbd>
   ```

5. 引用

   ```
   >引用内容
   ```

6. 条目

   ```
   1. 内容
   5. 无序输入，有序排列
   6. 按回车自动下一条
   8. 按‘回车’加‘Backspace’该条目内换行
   9. 按两次‘回车’跳出该条目
   
   * 无序排列
   + 亦可
   ```

7. 链接

   ```
   [显示](链接或路径)
   ```

   

8. 感觉用鼠标代替更加方便可还是知道了的语法

   * 表格

     | 233  | 233  | 2333 |
     | ---- | ---- | ---- |
     |      |      |      |

     ```
     |233|2333|23333|
     ```

     直接用插入更方便

   * 图片

     直接拖进去就好了，或者复制进去
