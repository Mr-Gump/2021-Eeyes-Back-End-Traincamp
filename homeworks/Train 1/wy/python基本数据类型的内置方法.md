## 二、序列：

##### 共有内置：

```python
len()	统计长度/个数
max()	min()	sum()

```

#### 字符串`string`

```python
split	分割字符串
title	将首字母大写
lower	所有字母小写
upper	所有字母大写
.replace(x,y)	用y替换x
```

#### 列表`list`

```python
del 	.pop()	.remove() 删除列表元素
.append	在末端增加元素
.insert 插入元素
.index	返回元素位置
.sort	升序排序
.count	返回元素出现次数
```
