# 4.手动实现`mysplit`函数,要求具有和`split`完全一样的功能


```python
def my_split(string, character):
    a = list(string)  # 把字符串分割成单个字符
    als = ""
    res = []
    for i in string:
        if i == character:
            res.append(als)
            als = ""
        else:
            als = als + i
    res.append(als)
    return res
```
