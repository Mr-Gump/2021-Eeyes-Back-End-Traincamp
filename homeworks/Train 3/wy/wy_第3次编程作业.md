## 2.随便给一个文件夹路径统计里面文件种类和对应数量


```python
import os
from pathlib import Path

Type=dict()
#输入要浏览的文件夹路径
File = input("文件目录")
#读取文件的后缀名
for root, dirs, files in os.walk(File):
    for i in files:
        f =Path(i).suffix  
        if f in Type: 
            Type[f]=Type[f]+1 
        else:
            Type[f]=1
            
name=Type.keys()
for i in name:
    print(i,':',Type[i])
```
